A — Centered Overlay

Classic command palette centered top of window. Dims background. Shows project/task type badges and parent project for tasks.

Pros

  • Familiar pattern (VS Code, Spotlight)
  • Maximum width for search results
  • Clear visual hierarchy

Cons

  • Covers the content area

B — Sidebar-Anchored

Search replaces the sidebar content, keeping the main content area fully visible. Natural extension of existing sidebar navigation.

Pros

  • Content stays visible while searching
  • Feels integrated with existing sidebar
  • Natural spatial location for navigation

Cons

  • Narrower search results area
  • Less visually prominent

C — Top Bar Dropdown

Search bar appears at top of the window with results dropping down. Similar to browser address bar or Raycast.

Pros

  • Compact and unobtrusive
  • Familiar from browser URL bars

Cons

  • Pushes content down or overlaps it
  • Less standard for desktop apps
