How should tab number hints appear on CMD hold?

These hints would fade in when you hold the Command key and fade out when you release it

1 > zsh ×
2 ✦ Claude ×
3 ◉ Browser ×

A — Corner Badge

Small numbered badge in the top-left corner of each tab. Accent colored, compact.

⌘1 > zsh ×
⌘2 ✦ Claude ×
⌘3 ◉ Browser ×

B — Inline Shortcut Pill

Shows "⌘1", "⌘2" etc. as a subtle pill before the tab icon. Matches the existing ⌘P hint style.

1 zsh ×
2 Claude ×
3 Browser ×

C — Replace Icon with Number

The tab icon (>, ✦, ◉) swaps to the tab number while CMD is held. Minimal, no extra space needed.

Right-pane tabs

For the right split pane, the hint would show ⇧1, ⇧2 etc. (since the shortcut is Cmd+Shift+N) using the same visual style as the left pane.
